== Description ==

An extension to Groups WooCommerce.

With this plugin enabled, when a subscription expires, instead of removing the group membership for the related user account, it will be kept.

Requires:
- [Groups](https://wordpress.org/plugins/groups/)
- [WooCommerce](https://wordpress.org/plugins/woocommerce/)
- [Groups WooCommerce](https://woocommerce.com/products/groups-woocommerce/)
